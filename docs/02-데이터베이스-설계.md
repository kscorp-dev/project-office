# 데이터베이스 설계서

## 1. ERD 개요

전체 테이블은 기능 모듈별로 그룹핑되며, `users`와 `departments` 테이블이 모든 모듈의 기반이 된다.

---

## 2. 공통 컬럼 규칙

모든 테이블에 아래 컬럼을 공통 적용한다:

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID (PK) | 기본 키 |
| created_at | TIMESTAMP | 생성 일시 (DEFAULT NOW()) |
| updated_at | TIMESTAMP | 수정 일시 (트리거 자동 갱신) |

soft delete가 필요한 테이블은 `deleted_at TIMESTAMP NULL`을 추가한다.

## 4. 인덱스 설계

### 주요 인덱스
```sql
-- 사용자
CREATE INDEX idx_users_email ON users(email);
CREATE INDEX idx_users_department ON users(department_id);
CREATE INDEX idx_users_status ON users(status);

-- 결재
CREATE INDEX idx_approval_docs_drafter ON approval_documents(drafter_id);
CREATE INDEX idx_approval_docs_status ON approval_documents(status);
CREATE INDEX idx_approval_docs_drafted ON approval_documents(drafted_at);
CREATE INDEX idx_approval_lines_approver ON approval_lines(approver_id, status);
CREATE INDEX idx_approval_lines_document ON approval_lines(document_id, step);

-- 메신저
CREATE INDEX idx_messages_room ON messages(room_id, created_at);
CREATE INDEX idx_messages_sender ON messages(sender_id);
CREATE INDEX idx_chat_room_members_user ON chat_room_members(user_id);

-- 게시판
CREATE INDEX idx_posts_board ON posts(board_id, created_at);
CREATE INDEX idx_posts_author ON posts(author_id);
CREATE INDEX idx_comments_post ON comments(post_id, created_at);

-- 근태
CREATE INDEX idx_attendance_user_date ON attendance_records(user_id, date);

-- 캘린더
CREATE INDEX idx_calendar_events_creator ON calendar_events(creator_id);
CREATE INDEX idx_calendar_events_range ON calendar_events(start_datetime, end_datetime);

-- 알림
CREATE INDEX idx_notifications_user ON notifications(user_id, is_read, created_at);

-- 감사 로그
CREATE INDEX idx_audit_logs_user ON audit_logs(user_id, created_at);
CREATE INDEX idx_audit_logs_entity ON audit_logs(entity_type, entity_id);

-- 작업명세서
CREATE INDEX idx_task_orders_creator ON task_orders(creator_id);
CREATE INDEX idx_task_orders_status ON task_orders(status);
CREATE INDEX idx_task_orders_due ON task_orders(due_date);
CREATE INDEX idx_task_assignees_user ON task_assignees(user_id, status);

-- 재고
CREATE INDEX idx_inventory_items_category ON inventory_items(category_id);
CREATE INDEX idx_inventory_items_sku ON inventory_items(sku);
CREATE INDEX idx_inventory_tx_item ON inventory_transactions(item_id, transaction_date);
CREATE INDEX idx_inventory_tx_date ON inventory_transactions(transaction_date);

-- 화상회의
CREATE INDEX idx_meetings_host ON meetings(host_id);
CREATE INDEX idx_meetings_status ON meetings(status, started_at);
CREATE INDEX idx_meeting_participants_user ON meeting_participants(user_id);
CREATE INDEX idx_meeting_minutes_meeting ON meeting_minutes(meeting_id);
CREATE INDEX idx_meeting_transcripts_meeting ON meeting_transcripts(meeting_id, timestamp_start);
```
