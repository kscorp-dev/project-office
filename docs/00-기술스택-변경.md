# 기술 스택 변경: React Native 통합

## 변경 사항

### Before (기존)
| 플랫폼 | 기술 |
|--------|------|
| Web | React + TypeScript + Vite |
| iOS | Swift + SwiftUI (네이티브) |
| Android | Kotlin + Jetpack Compose (네이티브) |

### After (변경)
| 플랫폼 | 기술 |
|--------|------|
| Web | React + TypeScript + Vite |
| Mobile (iOS/Android) | React Native + TypeScript + Expo |

### 변경 이유
1. **코드 재사용**: Web과 Mobile 간 비즈니스 로직, 타입, API 서비스 공유
2. **개발 효율**: 하나의 코드베이스로 iOS/Android 동시 지원
3. **유지보수**: 단일 언어(TypeScript)로 전체 스택 통합
4. **빠른 배포**: Expo EAS Build로 OTA 업데이트 가능

## 최종 기술 스택

### Backend
| 구분 | 기술 |
|------|------|
| Runtime | Node.js 20+ |
| Framework | Express.js |
| Language | TypeScript |
| ORM | Prisma |
| DB | PostgreSQL 15+ |
| Cache | Redis 7+ |
| 실시간 | Socket.IO |
| 인증 | JWT (Access + Refresh) |
| 파일 | AWS S3 / Local |
| 미디어 | FFmpeg (CCTV HLS) |

### Web Frontend
| 구분 | 기술 |
|------|------|
| Framework | React 18 |
| Language | TypeScript |
| Build | Vite |
| 상태관리 | Zustand |
| HTTP | Axios |
| 라우팅 | React Router v6 |
| UI | Tailwind CSS + shadcn/ui |
| 폼 | React Hook Form + Zod |
| 실시간 | Socket.IO Client |
| 차트 | Recharts |

### Mobile (React Native)
| 구분 | 기술 |
|------|------|
| Framework | React Native 0.76+ |
| Platform | Expo SDK 52+ (Managed → Bare) |
| Language | TypeScript |
| 네비게이션 | Expo Router (파일 기반) |
| 상태관리 | Zustand (Web과 공유) |
| HTTP | Axios (Web과 공유) |
| UI | React Native Paper + NativeWind |
| 폼 | React Hook Form + Zod (Web과 공유) |
| 실시간 | Socket.IO Client |
| 영상 | expo-av (HLS 네이티브 지원) |
| 푸시 | expo-notifications (APNs + FCM) |
| 보안 저장 | expo-secure-store |
| 생체인증 | expo-local-authentication |
| 카메라 | expo-camera (바코드 스캔 등) |
| 위치 | expo-location (GPS 출퇴근) |

### 공유 코드 (Web + Mobile)
| 구분 | 내용 |
|------|------|
| Types | API 요청/응답 타입, 도메인 모델 |
| API Services | Axios 기반 API 호출 함수 |
| Validation | Zod 스키마 (폼 유효성) |
| Utils | 날짜, 포맷, 계산 유틸리티 |
| Constants | 상수, Enum 값 |
| Hooks | 비즈니스 로직 훅 (UI 무관한 것) |

## 프로젝트 구조 (모노레포)

```
project-office/
├── apps/
│   ├── backend/          # API 서버 (Node.js + Express)
│   ├── web/              # 웹 프론트엔드 (React + Vite)
│   └── mobile/           # 모바일 앱 (React Native + Expo)
├── packages/
│   └── shared/           # 공유 코드 (Types, API, Validation)
├── docs/                 # 기획 문서
├── package.json          # 루트 (Turborepo)
└── turbo.json            # 모노레포 빌드 설정
```
