# 기능 명세서: 근태관리

## 1. 개요

직원의 출퇴근 기록, 근무시간 관리, 휴가 관리를 통합 처리하는 모듈이다.
GPS/IP 기반 위치 인증, 주 52시간 근무제 대응, 연차 자동 관리 기능을 포함한다.

---

## 2. 기능 상세

### 2.1 출퇴근 체크

**사용자 시나리오:**
1. 앱/웹 접속
2. 메인 화면의 "출근" 버튼 클릭
3. 위치/IP 인증 (설정된 경우)
4. 출근 기록 완료
5. 퇴근 시 "퇴근" 버튼 클릭

**상세 요구사항:**

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| ATT-001 | 출근/퇴근 버튼으로 시간 기록 | 필수 |
| ATT-002 | 외출/복귀 기록 | 중요 |
| ATT-003 | GPS 위치 인증 (반경 설정 가능) | 필수 |
| ATT-004 | IP 기반 인증 (회사 네트워크) | 중요 |
| ATT-005 | 출근 시간 표시 및 근무시간 실시간 카운트 | 필수 |
| ATT-006 | 지각 자동 판단 (설정 시간 기준) | 필수 |
| ATT-007 | 조퇴 자동 판단 | 필수 |
| ATT-008 | 출퇴근 미체크 알림 (출근 10분 후, 퇴근 시간 경과 시) | 중요 |
| ATT-009 | 출퇴근 기록 수정 요청 (관리자 승인 필요) | 중요 |
| ATT-010 | 오늘의 출퇴근 상태 대시보드 표시 | 필수 |

### 2.4 휴가 관리

**휴가 신청 시나리오:**
1. "휴가 신청" 메뉴 진입
2. 휴가 유형 선택 (연차, 반차, 병가 등)
3. 기간 선택
4. 사유 입력
5. 신청 → 전자결재 자동 연동

**상세 요구사항:**

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| VAC-001 | 휴가 유형: 연차, 오전반차, 오후반차, 병가, 특별휴가, 대체휴가 | 필수 |
| VAC-002 | 휴가 기간 선택 (시작일~종료일) | 필수 |
| VAC-003 | 사용 일수 자동 계산 (주말/공휴일 제외) | 필수 |
| VAC-004 | 잔여 연차 확인 | 필수 |
| VAC-005 | 잔여 연차 부족 시 신청 불가 | 필수 |
| VAC-006 | 휴가 신청 → 전자결재 휴가신청서 자동 생성 | 필수 |
| VAC-007 | 결재 승인 시 → 연차 자동 차감 | 필수 |
| VAC-008 | 결재 승인 시 → 캘린더 자동 등록 | 중요 |
| VAC-009 | 결재 승인 시 → 해당일 근태 '휴가' 처리 | 필수 |
| VAC-010 | 휴가 취소 신청 (결재 필요) | 중요 |
| VAC-011 | 휴가 취소 승인 시 → 연차 자동 복원 | 중요 |

### 2.5 연차 관리

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| ANN-001 | 입사일 기준 연차 자동 생성 (근로기준법 기준) | 필수 |
| ANN-002 | 연차 발생 현황 조회 (총 연차, 사용, 잔여) | 필수 |
| ANN-003 | 연차 사용 내역 조회 | 필수 |
| ANN-004 | 관리자 연차 수동 조정 (특별 부여/차감) | 중요 |
| ANN-005 | 연차 촉진 알림 (미사용 연차 소멸 전 알림) | 선택 |

**연차 자동 생성 규칙 (근로기준법):**
- 입사 1년 미만: 매월 1일 발생 (최대 11일)
- 입사 1년 이상: 연 15일
- 3년 이상 근속 시: 2년마다 1일 추가 (최대 25일)

## 3. 비즈니스 규칙

1. 출근 기록은 하루에 1건만 가능하다 (중복 출근 방지)
2. 출근 없이 퇴근 체크는 불가하다
3. 위치 인증 활성화 시, 허용 범위 밖에서는 출근 체크 불가
4. 근태 기록 수정은 관리자 승인이 필요하다
5. 공휴일/주말은 근무일에서 자동 제외한다
6. 휴가와 출근이 동시에 존재할 수 없다 (해당일 휴가 시 출근 체크 불필요)
7. 지각/조퇴 판단은 설정된 출퇴근 시간 기준으로 자동 처리한다
8. 근태 데이터는 최소 3년간 보관한다
