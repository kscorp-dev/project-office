# 기능 명세서: 문서관리 / 관리자 콘솔

---

# Part 1: 문서관리

## 1. 개요

개인/공유 문서함을 통해 파일을 체계적으로 관리하고 공유하는 기능이다.
버전 관리, 외부 공유 링크, 파일 미리보기를 지원한다.

---

## 2. 기능 상세

### 2.1 문서함

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| DOC-001 | 개인 문서함 (본인만 접근) | 필수 |
| DOC-002 | 부서 공유 문서함 (부서원 접근) | 필수 |
| DOC-003 | 전사 공유 문서함 | 중요 |
| DOC-004 | 폴더 생성/수정/삭제/이동 | 필수 |
| DOC-005 | 폴더별 접근 권한 설정 (읽기/쓰기/관리) | 필수 |
| DOC-006 | 파일 업로드 (드래그&드롭, 다중 선택) | 필수 |
| DOC-007 | 파일 다운로드 (개별, 폴더 일괄 ZIP) | 필수 |
| DOC-008 | 파일 이동/복사 | 필수 |
| DOC-009 | 파일 삭제 (휴지통 → 30일 보관) | 필수 |

### 2.2 파일 미리보기

|----------|-------------|---------|

| 텍스트 (TXT, CSV, JSON) | 텍스트 표시 | 필수 |
| 동영상 (MP4, MOV) | 비디오 플레이어 | 선택 |

---

## 3. 비즈니스 규칙

1. 개인 문서함 용량: 기본 5GB (관리자 조정 가능)
2. 부서 문서함 용량: 기본 20GB
3. 파일 최대 크기: 100MB
4. 삭제된 파일은 휴지통에 30일 보관 후 영구 삭제
5. 버전은 최대 20개까지 유지 (초과 시 가장 오래된 버전 자동 삭제)
6. 외부 공유 링크는 최대 30일 유효

# Part 2: 관리자 콘솔

## 1. 개요

시스템 관리자가 전체 플랫폼을 관리하는 통합 관리 화면이다.
**슈퍼관리자는 모든 기능 모듈을 개별로 ON/OFF 할 수 있다.**

---

## 2. 기능 상세

**데이터 모델:**

```
feature_modules 테이블
├── id: UUID
├── module_key: VARCHAR(50) UNIQUE  -- 'approval', 'messenger', 'cctv' 등
├── module_name: VARCHAR(100)       -- '전자결재', '메신저' 등
├── is_enabled: BOOLEAN             -- ON/OFF
├── is_core: BOOLEAN                -- true면 비활성화 불가 (조직도, 관리자콘솔)
├── disabled_by: UUID FK → users    -- 비활성화한 관리자
├── disabled_at: TIMESTAMP          -- 비활성화 시점
└── updated_at: TIMESTAMP
```

## 3. 슈퍼관리자 (Super Admin) 권한 체계

**역할 계층 확장:**

| 역할 | 코드 | 설명 |
|------|------|------|
| 슈퍼관리자 | super_admin | 기능 모듈 ON/OFF, 시스템 설정, 모든 관리 기능 |
| 시스템 관리자 | admin | 사용자/부서 관리, 보안 설정, 감사 로그 |
| 부서 관리자 | dept_admin | 소속 부서 관리 |
| 일반 사용자 | user | 기본 기능 사용 |
| 게스트 | guest | 제한적 기능 |

**슈퍼관리자 전용 기능:**
- 기능 모듈 ON/OFF
- 시스템 관리자 지정/해제
- 시스템 초기 설정
- 데이터 백업/복원 요청
- 전체 사용자 비밀번호 정책 강제 적용

---

## 4. 비즈니스 규칙

1. 슈퍼관리자는 시스템에 최소 1명 존재해야 한다
2. 조직도와 관리자콘솔은 비활성화할 수 없다 (is_core = true)
3. 기능 비활성화 시 해당 기능의 API 엔드포인트도 차단된다
4. 기능 비활성화 시 진행 중인 작업 (결재 진행중, 회의 중 등)은 완료 후 차단
5. 감사 로그는 삭제할 수 없다
6. 관리자 콘솔 접근 자체를 로깅한다
7. 비밀번호 초기화 시 사용자에게 이메일 발송
8. 스토리지 용량 90% 초과 시 슈퍼관리자에게 알림
